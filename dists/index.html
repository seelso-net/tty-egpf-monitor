APT Repository - Distributions

This APT repository supports both Ubuntu LTS versions with automatic compatibility handling.

Available Ubuntu Distributions:

  • jammy/ - Ubuntu 22.04 LTS (Jammy Jellyfish)
    • Automatic libbpf upgrade in postinst if needed
    • Compatible with both kernel 5.15.x and 6.8.x
  • noble/ - Ubuntu 24.04 LTS (Noble Numbat)
    • Uses native libbpf 1.7.0+
    • No special handling required
