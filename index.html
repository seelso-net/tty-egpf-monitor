tty-egpf-monitor APT Repository

Installation

Add this repository to your system:

# Add GPG key (using modern signed-by method)
wget -O - https://seelso-net.github.io/tty-egpf-monitor/public-apt-key.asc | gpg --dearmor | sudo tee /usr/share/keyrings/tty-egpf-monitor-archive-keyring.gpg > /dev/null

# Add repository with signed-by option
echo "deb [signed-by=/usr/share/keyrings/tty-egpf-monitor-archive-keyring.gpg] https://seelso-net.github.io/tty-egpf-monitor $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/tty-egpf-monitor.list

# Update and install
sudo apt-get update
sudo apt-get install tty-egpf-monitord tty-egpf-monitor-cli

Available Packages

  • tty-egpf-monitord - eBPF serial monitoring daemon with HTTP API
  • tty-egpf-monitor-cli - CLI client for managing the daemon

Repository Structure

  • dists/ - Distribution metadata
  • pool/ - Package files
  • public-apt-key.asc - GPG public key

Supported Distributions

  • Ubuntu 22.04 (jammy) - amd64, i386
  • Ubuntu 24.04 (noble) - amd64, i386
