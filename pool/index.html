APT Repository - Package Pool

Package files (.deb) organized by Ubuntu distribution. Each directory contains all available versions.

Jammy packages: $(ls jammy/*.deb 2>/dev/null | wc -l) files

Noble packages: $(ls noble/*.deb 2>/dev/null | wc -l) files

Package Directories:

  • jammy/ - Ubuntu 22.04 LTS packages (.deb files)
    • Includes automatic libbpf compatibility in postinst
    • All package versions available for download
  • noble/ - Ubuntu 24.04 LTS packages (.deb files)
    • Uses native libbpf, no special handling
    • All package versions available for download
